# nineb1ts.de

> curiosity is compiling...

Personal portfolio and digital playground of **nineb1ts**.

The site is built with React, TypeScript and Vite and deployed automatically to GitHub Pages.

<br>
<br>

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- GitHub Actions
- GitHub Pages

## Development

Install dependencies:

```bash
npm install
```

Start the local development server:

```bash
npm run dev
```

## Build

Create a production build:

```bash
npm run build
```

## Deployment

Deployments are handled automatically through GitHub Actions whenever changes are pushed to `main`.
